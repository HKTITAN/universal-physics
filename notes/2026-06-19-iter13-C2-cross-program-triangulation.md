# Iteration 13 — C2: Cross-Program Triangulation

Outcome: TRIANGULATION-INVARIANT-FOUND (one genuine shared invariant), but VERDICT-NEUTRAL — triangulation re-confirms the encode screen rather than breaking it. Frame/synthesis audit, NOT a carrier re-attack. Carrier count FIVE; no hedge moves; no new IDs.

## The question
Do the major unification programs share any ROBUST structural feature that any final theory must satisfy, independent of which program is right — and does it bear on the encode-not-generate / not-forecastable verdict?

## Fundamental-vs-derived map (web-verified 2024-2026)
- String / AdS-CFT: boundary CFT fundamental; bulk metric + graviton emergent (metric-from-information). [SPECULATIVE as nature]
- Asymptotic safety: metric stays an ordinary QFT field; UV (Reuter) fixed point fundamental; NO emergence; signature installed by hand. [OPEN]
- Causal sets: discrete Lorentzian partial order fundamental; continuum metric emergent (order+number=geometry). [OPEN]
- LQG / spin foams: quantized discrete geometry fundamental; continuum emergent as semiclassical limit. [OPEN]
- Entanglement / it-from-qubit (Program A): algebra+state (modular) fundamental; geometry encoded, ENCODES-not-GENERATES (standing verdict). [SPECULATIVE]

## The one genuine shared invariant: area law S=A/4G
Referee-survivable: multiply-derived, web-verified, accepted as a consistency testbench by all five programs (Strominger-Vafa; LQG horizon punctures; causal-set element count; Ryu-Takayanagi; plus a general canonical-quantization statement). But verdict-neutral by construction: a retrodiction of a semiclassical result that every program AND every geometry-first description reproduces, so it is screen-passing (HYP-ENCODING-SCREEN), not class-separating. It is the entropy-side shadow of the missing reverse-Weinberg-Witten observable — re-deriving, from the program-comparison direction, the iter-9 identity reverse-WW = carrier.

## Candidate invariants that FAIL the referee (shared vocabulary / open conjecture)
- UV dimensional reduction d_s to 2: method-dependent; in causal sets the random-walk spectral dimension INCREASES at short scales — opposite sign. Shared label, different computations.
- Holographic finiteness of information: genuine inside the string/swampland/it-from-qubit camp, but asymptotic safety conflicts with holography + statistical BH entropy (arXiv:2502.12290). Camp-local.
- Background independence: central for LQG/causal sets/CDT; unrealized in string/M-theory. Shared open problem.
- Geometry-is-emergent: fractures into continuum-from-discreteness vs metric-from-information (different substrates), with asymptotic safety dissenting. Ontological mood.

Common failure mode: a WORD is shared while content/substrate/commitment differs or is contested. A shared vocabulary is not a shared prediction; a popular conjecture is not an established constraint.

## Does it bear on the verdict? No — and why
1. The shared ontological leaning SUPPORTS Program A's ontology but changes nothing: ontological agreement is exactly what the encode screen concedes and neutralizes (all programs encode the same semiclassical EFT — all reproduce S=A/4, GR-as-EFT, the SM where they reach it), so it buys no distinguishing observable. Triangulation lands INSIDE the screen.
2. The one genuine invariant is screen-passing, not class-separating: a triangulation-derived test would discriminate only if realizable without a fundamental metric (the reverse-WW property), and the area law is not one (defined ON the horizon geometry). No such cross-program observable exists as of 2026-06.
3. Experimental corroboration: BMV/QGEM tests the gate, cannot settle quantization, does not distinguish string vs loop; standard Planck-scale arguments are individually weak (arXiv:2501.07614).

Net: a real positive (the area-law invariant) and a real negative (no class-separating shared prediction) together TIGHTEN the encode-not-generate / not-forecastable verdict rather than moving it.

## Honest residue
The general canonical-quantization-satisfies-S=A/4 claim is itself conditional (low-T, large-area, a large class of dynamics) — which only strengthens the point that even the one invariant is a semiclassical-regime consistency statement, not a Planck-regime prediction. The AS-vs-swampland conflict is one defensible reading (single conceptual-assessment paper), not consensus; either reading supports the camp-local conclusion. A class-separating shared prediction could exist unnoticed; the claim is the conservative literature-state one (none on the table), a literature fact not a proven impossibility.
